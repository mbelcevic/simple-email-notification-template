This is very simple email template for different (company) updates.

It consists of a 7 parts, and has 6 classes.


1) Header with centred logo (class "email-header")

2) Neutral paragraph (class "email-body")

3) Segment for the updated part, highlighted (class "email-updates")

4) Neutral paragraph (class "email-body")

5) Signature (class "email-signature")

6) Company info with links (class "email-companyinfo")

7) Terms/Privacy statement (class "email-terms")

The whole email (all 7 parts) is packed into a division that has class "email-container", which defines the basic style (e.g. font family) for the whole email.
